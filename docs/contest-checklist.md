## 已完成

- React + TypeScript 前端工作台。
- Express 本地 API 服务。
- 场景选择：机场值机、面试自我介绍、咖啡店点单、项目路演开场。
- 多模式能力：场景陪练、自由对话、语法表达诊断、学习计划、词汇短语库。
- 语音/文字输入。
- 浏览器朗读教练回复，形成端到端语音对话体验。
- OpenAI-compatible Chat Completions 接入。
- 发音评测：发音分、识别置信度、发音建议。
- 语法/表达纠错：错误点、建议写法、原因说明。
- 课后总结：下一步重点和 homework。
- 量化反馈：总分、流利度、发音、语法、词汇、互动性、延迟、语速、词数。
- 口语反馈接口：分数、水平、亮点、改进建议、改写版本、下一轮追问。
- 无模型密钥时的本地兜底反馈。
- README、环境变量模板、提交说明。
- 公开 GitHub 仓库已创建。
- `main` 分支已推送到远程。
- `npm run lint` 和 `npm run build` 需要在每次提交前确认通过。
